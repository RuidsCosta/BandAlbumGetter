De modo a automatizar a procura das imagens das capas dos albums, assim como os seus nomes e datas de criação, de cada banda, criei um pequeno programa em python que, ao introduzirmos uma lista com os vários URI's de Spotify de cada banda, para cada elemento dessa lista vai buscar o nome da banda, criando uma pasta com esse nome e para essa mesma banda faz o download das imagens de todos os seus albums e coloca-os na pasta da respetiva banda com  o formato "nome_data.png". Isto tudo é possivel com uma libraria para python que nos dá acesso à API do Spotify, o Spotipy. Com esta podemos fazer varias operações sobre a base de dados do spotify e manipular os dados que esta possui.
